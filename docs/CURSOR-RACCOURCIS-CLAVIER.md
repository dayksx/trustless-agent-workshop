# Raccourcis clavier Cursor (Linux)

> Cursor hérite des raccourcis de VS Code. Pour voir ou modifier les raccourcis : **Ctrl+K** puis **Ctrl+S**, ou *Fichier > Préférences > Raccourcis clavier*.

---

## Édition de base

| Action | Raccourci |
|--------|-----------|
| **Renommer** une variable/fonction/élément partout dans le fichier | **F2** |
| **Dupliquer** une ligne en dessous | **Shift+Alt+↓** |
| **Dupliquer** une ligne au-dessus | **Shift+Alt+↑** |
| **Déplacer** une ligne vers le bas | **Alt+↓** |
| **Déplacer** une ligne vers le haut | **Alt+↑** |
| **Supprimer** la ligne courante | **Ctrl+Shift+K** |

---

## Sélection multiple et curseurs

| Action | Raccourci |
|--------|-----------|
| **Sélectionner la prochaine occurrence** du mot (multi-curseur) | **Ctrl+D** |
| **Sélectionner toutes les occurrences** du mot | **Ctrl+Shift+L** |

---

## Refactoring et navigation

| Action | Raccourci |
|--------|-----------|
| **Actions rapides / Refactoring** (extraction, correction, etc.) | **Ctrl+.** |
| **Refactoring uniquement** | **Ctrl+Shift+R** |
| **Aller à la définition** | **F12** ou **Ctrl+clic** |
| **Aperçu de la définition** (sans quitter le fichier) | **Alt+F12** |

---

## Commentaires

| Action | Raccourci |
|--------|-----------|
| **Commenter / décommenter** une ligne | **Ctrl+/** |
| **Commenter un bloc** | **Shift+Alt+A** |

---

## Recherche et remplacement

| Action | Raccourci |
|--------|-----------|
| **Rechercher dans le fichier** | **Ctrl+F** |
| **Rechercher dans tout le projet** | **Ctrl+Shift+F** |
| **Remplacer** | **Ctrl+H** |

---

## Formatage et commandes

| Action | Raccourci |
|--------|-----------|
| **Formatage du document** | **Shift+Alt+F** |
| **Palette de commandes** | **Ctrl+Shift+P** |
| **Afficher tous les raccourcis** | **Ctrl+K** puis **Ctrl+S** |

---

## Raccourcis spécifiques Cursor (IA)

| Action | Raccourci |
|--------|-----------|
| **Ouvrir le chat Cursor** | **Ctrl+L** |
| **Composer** (édition multi-fichiers) | **Ctrl+I** |
| **Inline Edit** (éditer avec l'IA dans le code) | **Ctrl+K** |
